Markdown is great but limited. qdoc is a small collection of addons for the Markdown converter Pandoc to extend the possibilities of Markdown. In particular, qdoc adds

  • True Multi-Output: You can convert the same source file to html, gfm, or pdf and it will look great.
    • Math Support: You can typeset math in tex and it will be rendered for every output.
    • Icon Support: Use unicode icons that will be rendered with FontAwesome when available and otherwise fallback to native icons.
    • No Artefacts: If a feature is not supported for one output type, it will not cause troubles and rendered in a simpler form.
  • Additional Syntax:
    • Inline Diagrams: You can embedd code for diagrams in the .md and qdoc will render the image. Currently, we support matplotlib, ggplot, tikz, plantuml
    • Colored Boxes: You can add boxes to seperate special content from the normal text flow.
    • Collapse Details: Hide additional info by default and reveal it only when the user clicks on it.
    • Include Files: You can include the content of other .md files.
  • Awesome HTML: If the output is html, qdoc will add bonus features to enhance readability.
    • Dark Mode: You can read the content in dark theme.
    • Adjust Font Size: You can increase or decrease the font size.
    • Print Friendly: Looks nice on A4 paper when printed.
    • Heading Anchors: automatic anchors at h2 headings for one-click links.

Getting Started

Only 4 small steps are required to build your first document.

1. Clone this repo.

git clone git@github.com:emareg/qdoc.git

2. open and edit my-first-template.md

xdg-open src/my-first-template.md

3. build the file using make:

make

4. open build/my-first-template.html in a browser

Math Support

Math is well supported for every output. For HTML we render math via KaTeX. If you read this on GitHub, math will be rendered as an svg image. For PDF, math is supported natively by LaTeX.

There is inline math like \sqrt{x^2 + \frac{1}{2}} and display math like

f_X(x)=\frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}

The equation above is created by typesetting

$$f_X(x)=\frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

Boxes and Cards

To mark special content, you may use boxes. Simply start a line with KEY:, where KEY is a keyword from one of the categories:

  • Hints/Examples: Example, Hint, Tipp
  • Notes: Note, Notice, Info
  • Warning: Warn, Warning, Danger, Caution

For example if write a paragraph like

Note: This is a simple note box.
